Required Fields can check if you have fill in all the fields you have enabled
if you forgotten something it reminds you that you have forgot something

== Description ==

You Can enable Categories, Tags, thumbnail Validator, if you have forgotten to add Categories or Tags Or Thumbnail Image We can remind you With an simple Error

== Installation ==

1. Extract the `Required-Fields` folder to your `wp-content/plugins` directory
2. Activate the plugin through the admin interface

== Frequently Asked Questions ==

= Are there any settings I can adjust?

= Nope! Just Click ON For the option you want to alert when you forgotten something
